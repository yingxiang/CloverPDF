Technical Support

Help with WPDF

Troubleshooting guidance for PDF merging, file conversion, OCR, purchases, and saved results on macOS.

What WPDF supports

WPDF merges multiple PDF files into one PDF, PNG, JPG, or Word file. It can also convert individual or multiple PDFs to PDF, PNG, JPG, or Word while preserving selected pages.

Common questions

A PDF cannot be added or opened

Confirm that the file still exists and that WPDF has permission to access its folder. Add the file again if it was moved, renamed, or restored from external storage. Password-protected PDFs must be unlocked with the correct password.

Conversion fails

Make sure the destination folder is writable and the source PDF is not damaged. Try converting a smaller page selection. If a previous task lost file access after a move or rename, add the source PDF again and start a new conversion.

A scanned PDF produces poor Word text

Keep “OCR scanned documents” enabled in Settings for editable text. OCR quality depends on scan resolution, rotation, contrast, language, tables, stamps, and complex layouts. Disable OCR when you prefer the converter's direct handling of the scanned document.

The generated file cannot be found

Select the task and use “Show in Finder.” A successful task keeps its saved path, but WPDF cannot restore an output that was moved or deleted outside the app. Double-click a successful task to open its generated file.

A purchase is not recognized

Confirm that the Mac App Store is signed in with the Apple Account used for the purchase. Open the upgrade page and use Restore Purchases. StoreKit may require an internet connection to refresh transaction status.

Before requesting support

  • Install the latest WPDF and macOS updates available to you.
  • Restart WPDF and repeat the operation once.
  • Record the WPDF version shown in Settings.
  • Note the source format, selected output format, page count, and exact error message.
  • Do not send confidential source documents unless specifically requested and safe to share.

Contact support

Use the developer contact information on the WPDF App Store listing. Include your macOS version, WPDF version, the steps that reproduce the issue, and the full error message. Screenshots are helpful when they do not contain private document information.
